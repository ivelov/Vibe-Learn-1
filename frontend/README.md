# Kanban Board

A single-board Kanban app with drag-and-drop cards across five columns. In-memory only, no backend.

## Stack

Next.js (App Router, TypeScript), Tailwind CSS v4, @dnd-kit for drag-and-drop, Vitest + React Testing Library for unit tests, Playwright for end-to-end tests.

## Getting started

```
npm install
npm run dev
```

Open http://localhost:3000.

## Testing

```
npm run test      # unit tests
npm run e2e        # Playwright end-to-end tests
```
